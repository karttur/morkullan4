Ritningen nedan är komprimerad både i storlek och kvaltiet. För att ladda hem ritningen i fullt format kan du antingen:

1. Öppna arkivet som innehåller alla ritningar (på GitHub),

2. klicka direkt på figuren, ritningen i fullt format kommer då upp på skärmen och du kan dra den till skrivbordet,

3. högerklicka på figuren (eller länken i nästa punkt) och sedan välja "Spara länkad fil som .." ("Download Linked file As .."), eller liknande,

4. Klicka på den här länken till ritning i fullt format, och sedan spara ritningen.
